Lab 4: Everyday Task + Game Pseudocode

Challenge

In this lab, we pseudocoded an unfamiliar everyday task and a simple computer game. Then we converted the pseudocode into JavaScript comments and organized everything into a webpage.

Problems

Understanding how to break down tasks logically was a bit tricky at first, but breaking everything into high-level steps made it manageable.

Results

Everyday Task: Changing a Tire


1. Park the car safely
    a. Find a flat surface
    b. Turn on hazard lights
    c. Apply parking brake
2. Gather necessary tools
    a. Spare tire
    b. Jack
    c. Lug wrench
3. Loosen lug nuts slightly
4. Jack up the car
5. Remove lug nuts and take off the flat tire
6. Mount the spare tire
7. Tighten the lug nuts by hand
8. Lower the car using the jack
9. Fully tighten the lug nuts with the wrench
10. Put away tools and flat tire

Game Task: Rock-Paper-Scissors


1. Display game options to player (rock, paper, scissors)
2. Get player input
3. Randomly select computer choice
4. Compare player and computer choices
    a. If same, it’s a tie
    b. If player wins, display win message
    c. If computer wins, display lose message
5. Ask if player wants to play again
