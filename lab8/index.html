Lab 8 - Anon Functions and Callbacks

Challenge

The challenge of the lab was to design and implement a responsive webpage using HTML and CSS that adheres to accessibility standards and functions well on both desktop and mobile devices.

Problems

One issue was that my layout broke on smaller screens due to fixed-width elements. I solved this by using flexible units like percentages and adding media queries to adjust styles based on screen size. I also had trouble aligning elements, which I fixed using Flexbox.

Reflection

This assignment was challenging but rewarding. I spent a lot of time troubleshooting layout bugs and researching CSS best practices. I feel more confident in using responsive design techniques and learned the importance of planning before coding.
